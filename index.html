멋봉리 화석 좌표 계산기

네번의 탐지값으로 거의 정확한 화석 좌표를 얻어보자.

간단한 사용 예시

화석과의 거리가 50이하가 될때까지 이동하세요
탐지기를 작동시키고 작동위치를 기준 좌표에 입력하세요 (소수점은 모두 버리세요)
첫번째 상대좌표의 x,y,z에 모두 0을 입력하고 탐지된 거리값을 입력합니다.
x축으로 5칸 이동하고 다시 탐지기를 작동시키세요. 두번째 상대좌표에 5, 0, 0과 거리값을 입력하세요
다시 기준좌표로 돌아간다음 이번엔 z축으로 5칸 이동하고 다시 탐지기를 작동시킵니다. 세번째 상대좌표에 0,0,5, 탐지된 거리값을 입력합니다.
다시 기준좌표로 돌아간다음 이번엔 y축으로 5칸 이동하고 다시 탐지기를 작동시킵니다. 네번째 상대좌표에 0,5,0, 탐지된 거리값을 입력합니다.
계산 버튼을 눌러 좌표를 확인하세요. 계산에 실패하면 계산 허용 오차를 1씩 올려보면서 계산버튼을 누르세요. 4를 넘어서면 이상한 결과가 나오게 될지도 모릅니다.

다시 기준좌표로 돌아가는것이 귀찮다면 x축으로 5칸 이동하고 좌표와 거리를 입력한 후 z축으로 5칸 이동하고 좌표에 5,0,5, 거리값을 입력하면 됩니다.
거기서 y축으로 5만큼 더 이동해서 탐지했다면 5,5,5, 거리값을 입력하면 됩니다.
